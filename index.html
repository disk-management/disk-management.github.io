    • Disk Management

    Disk management is a Windows utility, allows to access and manage disk drives. Besides, it helps in drive partitioning, formatting, changing drive partition volume, initialising new disks and allocating a disk letter.



    Open Disk Management

    Follow steps given below to access Disk management:

    1. Right-click the Start button.
    2. Tap 'Disk Management'.
    OR

    1. Press Window+R.
    2. Now type 'diskmngmt.msc' in the Run dialog box.
